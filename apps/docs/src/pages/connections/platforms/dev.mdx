## How to connect?

<Steps>
### Step 1

Go to **My Account** -> **Settings** -> **Connections**.

### Step 2

Under **Platforms**, choose **Dev** and click **Connect**.

### Step 3

Enter your **API Key** and select default publish status, then click **Connect**.

### Done ✅

</Steps>

## How to get API Key?

<Steps>
### Step 1

Open Dev.to [extensions](https://dev.to/settings/extensions) page and scroll all the way down to **DEV Community API Keys**.

![DEV Community API Keys](/images/platforms/dev-1.png)

### Step 2

Generate a new API Key by clicking **Generate API Key**.

### Done ✅

</Steps>

Same way you can update details by clicking edit icon button.

If you would like to remove connection simple click **Disconnect**.
