# Ghost

import { Steps } from "nextra/components";

## How to connect?

<Steps>
### Step 1

Go to **My Account** -> **Settings** -> **Connections**.

### Step 2

Under **Platforms**, choose **Ghost** and click **Connect**.

### Step 3

Enter your **Admin API Key**, **API URL** and select default publish status, then click **Connect**.

### Done ✅

</Steps>

## How to get API Key and API URL?

<Steps>
### Step 1

Open your Ghost Admin dashboard -> Go to settings.

### Step 2

Search for **Integrations** and click on **Add custom integration**.

![Ghost integrations](/images/platforms/ghost-1.png)

### Step 3

Give your integration a name and click **Add**.

### Step 4

Copy the **Admin API Key**, **API URL** and click **Save & close**.

![credentials](/images/platforms/ghost-2.png)

### Done ✅

</Steps>

Same way you can update details by clicking edit icon button.

If you would like to remove connection simple click **Disconnect**.
