# Hashnode

import { Steps } from "nextra/components";

## How to connect?

<Steps>
### Step 1

Go to **My Account** -> **Settings** -> **Connections**.

### Step 2

Under **Platforms**, choose **Hashnode** and click **Connect**.

### Step 3

Enter your **API Key** and select default options, then click **Connect**.

### Done ✅

</Steps>

## How to get API Key?

<Steps>
### Step 1

Go to Hashnode [developer settings](https://hashnode.com/settings/developer) page and under "Personal Access Token", click **Generate new token**.

![Hashnode Developer Settings](/images/platforms/hashnode-1.png)

### Done ✅

</Steps>

Same way you can update details by clicking edit icon button.

If you would like to remove connection simple click **Disconnect**.
