# Medium

import { Steps } from "nextra/components";

## How to connect?

<Steps>
### Step 1

Go to **My Account** -> **Settings** -> **Connections**.

### Step 2

Under **Platforms**, choose **Medium** and click **Connect**.

### Step 3

Enter your **API Key** and select default options, then click **Connect**.

### Done ✅

</Steps>

## How to get API Key?

<Steps>
### Step 1

Go to Medium [settings](https://medium.com/me/settings/security) page and under "Security and apps", click **Integration tokens**.

![Medium Settings](/images/platforms/medium-1.png)

### Step 2

Give some description for your token and click **Get token**.

### Done ✅

</Steps>

Same way you can update details by clicking edit icon button.

If you would like to remove connection simple click **Disconnect**.
