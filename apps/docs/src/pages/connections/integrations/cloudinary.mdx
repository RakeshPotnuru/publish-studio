# Cloudinary

import { Steps } from "nextra/components";

## How to connect?

<Steps>
### Step 1

Go to **My Account** -> **Settings** -> **Connections**.

### Step 2

Under **Integrations**, choose **Cloudinary** and click **Connect**.

### Step 3

Enter your **API Key** and **Cloud name**, then click **Connect**.

### Done ✅

</Steps>

## How to get API Key and Cloud name?

<Steps>
### Step 1

Go to Cloudinary [developer dashboard](https://console.cloudinary.com/pm) page and under "Product Environment Credentials", you will find your **Cloud name** and **API Key**.

![Cloudinary developer dashboard](/images/integrations/cloudinary-1.png)

### Done ✅

</Steps>

Same way you can update details by clicking edit icon button.

If you would like to remove connection simple click **Disconnect**.
